# STM32 Smart Car Project

## Introduction

This project is an intelligent smart car developed with STM32.

## Functions

- Line Following
- Obstacle Avoidance
- Ultrasonic Distance Measurement
- PWM Motor Speed Control

## Hardware

- STM32
- HC-SR04
- Motor Driver
- DC Motors
- Infrared Sensors

## Software

- C Language
- Keil MDK

## Project Status

Completed as a college embedded systems project.
