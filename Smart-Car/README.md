# STM32 Smart Car

## Introduction
An intelligent vehicle project based on STM32.

## Features
- Line Tracking
- Ultrasonic Obstacle Avoidance
- PWM Motor Control
- Buzzer Alarm
- Key Control

## Hardware
- STM32F103
- Ultrasonic Sensor HC-SR04
- DC Motors
- Motor Driver

## Software
- C
- Keil MDK
- STM32 Standard Peripheral Library
