# STM32 ADC Light Sensor

## Introduction
This project demonstrates light intensity acquisition using the ADC peripheral of the STM32F103.

## Features
- ADC sampling
- Light sensor detection
- LED control

## Hardware
- STM32F103C8T6
- Light Sensor
- LED

## Development Environment
- Keil MDK
- C Language
- STM32 Standard Peripheral Library
